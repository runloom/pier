Pier · App 图标

三个停靠的方向

Pier 是本地 AI 开发工作台——名字本意「码头 / 栈桥」，正是产品的核心隐喻：多个 agent 在这里停靠、被看见、回到项目的连续性。三个方向都建立在品牌的紫罗兰主色与近黑底之上，克制到单一强调色，适配 macOS / Windows / Linux 桌面与 favicon。

A — 栈桥与倒影

The Pier

deck · pilings · reflection

侧看的栈桥：一道甲板横贯，三根桩柱探入水面，水线下泛起紫光倒影；顶端一盏泊灯是刚靠岸的 agent。直接画出产品的名字，叙事感最强、最难被认错。桩柱阵列在 16px 下仍读作「∏」形剪影。

Docked Panels

dockview · active panel · terminal

C — 泊位

The Berth

cradle · moored node · calm

最抽象、最安静的一支：一个 U 形泊位用单一笔触托起一颗发光的节点——被妥善停泊的 agent。大量留白、单线条、纯几何，缩放极其稳健，气质冷静克制，适合追求长期耐看的标志。
